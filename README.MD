# Bot Converter
Async bot converter with AioHttp and AioGram

## How to run
1. Clone the project
2. Install requirements.txt
```sh
pip install -r requirements.txt
```
3. Create and fill .env file
```
TELEGRAM_API_KEY=telegram_api_key
LOG_FILENAME=log_file_name
```

4. Run bot
```sh
python3 bot.py
```
